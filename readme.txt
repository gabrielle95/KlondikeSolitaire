Projekt IJA - Solitaire
-- Implementacia kartovej hry Solitaire v Java SE 8

Autori:
-- Gabriela Pacáková, xpacak01
-- Jozef Mikovič, xmikov01

Preklad:
-- ant compile

Spustenie:
-- ant run

O aplikacii:
-- Aplikacia pri spusteni obsahuje 1 hracie platno.
-- Pomocou ponuky menu ide spustit 1-4 hry sucastne. 
-- Kazda hra obsahuje prvok Undo - moznost navratenia, Save a Load.
-- Karty ide presuvat pomocou drag & dropu.
-- Uzivatel zvitazi, ked naplni 4 horne balicky kartami rovnakej farby od A po K.
-- Na 7 dolnych balikoch moze na seba klast len karty ktore nie su rovnakej farby.
-- Pre ziskanie dalsich kariet pouzivatel klika v lavo hore na balicek kariet s logom VUT.
